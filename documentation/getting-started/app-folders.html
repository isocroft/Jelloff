  • §/public

    The public folder contains the index.php entry file for the Jollof application as well as other static files needed to make the application work on the front-end. For instance, css, js, font folders contain the resources for the app. The sitemap xml file for informing crawlers of your web application structure and pages.

    Other files include the manifest json file which aids the process of turning your web app into an installbale on a mobile device. Also the manifest appcache file and offline html file for offline support.

    These are all provided to make your app development easy.

  • §/documentation

    The documentation folder contains all files that provide information on how to build web applications using Jollof.

  • §/system/routes

    The routes folder contains all the routes for the application. Routes can be defined and/or removed in the setup.php file within this folder.

  • §/system/middlwares

    The middlewares folder contains all application middlewares that run per request.
